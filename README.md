# Carrossel IstoPost Pro

Versão personalizada do criador de carrosséis de Instagram, do **João Paulo (IstoPost)**.

Instala a habilidade **carrossel-pro** no **Claude Code / Claude Cowork**: cria carrosséis prontos pra publicar e entrega o editor visual **Carousel Studio** já preenchido — pra ajustar textos, tipografia, cores, imagem de fundo e exportar os PNGs em ZIP.

## ✨ Personalizações desta versão

Além de tudo do editor original, esta versão acrescenta:

- 🎨 **Estilos prontos** — 12 presets (fonte + cores) com 1 clique
- ⭐ **Meus estilos** — salve seu visual e reaplique sempre
- 🏷️ **Minhas marcas** — guarde logo + @ + cores + fonte de cada marca
- 🅰️ **Cor automática do texto** — texto claro/escuro conforme o fundo
- 🔢 **Contador de caracteres** — avisa quando o texto pode estourar o slide
- 🌈 **Efeitos de fundo** — gradiente e brilho/spotlight com a cor de acento
- ↩️ **Desfazer / Refazer** (Ctrl+Z / Ctrl+Y)
- 📚 **Biblioteca de modelos** — 6 carrosséis-base prontos
- 🔡 **Entrada por código hexadecimal** — cole a cor exata da marca

## Instalar

### Claude Cowork (app desktop)
1. Configurações → Plugins → **Adicionar marketplace**.
2. Fonte **GitHub** → informe: `jpzimermann-glitch/carrossel-istopost-pro`
3. Instale o plugin **carrossel-istopost** no catálogo.

### Claude Code (terminal)
```bash
/plugin marketplace add jpzimermann-glitch/carrossel-istopost-pro
/plugin install carrossel-istopost@carrossel-istopost-pro
```

## Usar

Depois de instalar, peça: **"cria um carrossel sobre X"**. A habilidade conduz as perguntas, escreve a copy e entrega o editor já preenchido. Abra no navegador, ajuste e clique em **Exportar ZIP**.
